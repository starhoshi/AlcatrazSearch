## Getting Started

Requirement:

- NodeJS 4+

Install dependencies:

```
npm install
typings install
```

## Running development server

Run webpack dev server (for assets):

```
npm run dev-server
```

Visit [http://localhost:2992](http://localhost:2992)

## build for production

Output build/public:

```
npm run build
```


### Testing

To run tests, use:

```
npm test
```
